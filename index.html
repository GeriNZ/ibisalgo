Your mission? Guide the Scarlet Ibis through the maze of obstacles to reach the UWI logo safely and efficiently. With your help, this determined bird can overcome the barriers and get the education it deserves!

Beginners

Use only the two basic commands:

  • move: Moves the bird forward by one cell.
  • turn left: Turns the bird 90Â° counterclockwise.

Plan your steps carefully and use these commands to navigate through obstacles.

Intermediate

Enhance your code with control structures and egg commands:

  • For-loops: Repeat a block of commands a set number of times.
    for 3: (followed by indented commands) 
    for 3:
        turn left
        move
  • While-loops: Execute commands repeatedly as long as a condition is met.
    while front_clear: (followed by indented commands) 
    while front_clear:
        move
  • Egg Commands:
    • lay egg: Drop an egg on the current cell.
    • pick up egg: Remove an egg from the current cell (if one exists).
    Use these to mark your path or create patterns.

Expert

Combine custom functions with loops and egg commands to solve challenging puzzles:

  • Custom Functions: Define your own set of commands to simplify complex sequences.
    Define a function with: 
    function turn_right:
        turn left
        turn left
        turn left
    Then, call it in your main code by simply writing: 
    turn_right
  • Mix functions with loops and basic commands to create intricate patterns and sophisticated algorithms.
